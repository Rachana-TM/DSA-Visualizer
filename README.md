# 🌳 DSA Visualizer

An interactive **Data Structures and Algorithms (DSA) Visualizer** built using **HTML, CSS, and JavaScript**. This project helps students understand the working of different data structures and algorithms through visual animations and an intuitive user interface.

---

## 🚀 Features

### 📦 Data Structures
- ✅ Array
  - Create
  - Insert
  - Delete
  - Search
  - Update
  - Traverse

- ✅ Stack
  - Push
  - Pop
  - Peek
  - Size

- ✅ Queue
  - Enqueue
  - Dequeue
  - Front
  - Rear

- ✅ Linked List
  - Insert
  - Delete
  - Search
  - Traverse

- ✅ Tree
  - Binary Tree
  - Binary Search Tree (BST)
  - Insert
  - Delete
  - Search
  - Inorder Traversal
  - Preorder Traversal
  - Postorder Traversal
  - Level Order Traversal

- ✅ Graph
  - Add Vertex
  - Add Edge
  - Remove Edge
  - Breadth First Search (BFS)
  - Depth First Search (DFS)
